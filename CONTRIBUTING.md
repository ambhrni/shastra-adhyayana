# Contributing to Tattvasudhā

Tattvasudhā (tattvasudha.org) is a jñānayajña — a platform for the  study of Sanskrit śāstra texts in the Madhva Dvaita Vedānta
tradition. We welcome contributions from developers, Sanskrit scholars,
and devoted learners.

## Two types of contribution

### Technical contributions (code)
If you want to improve the platform — better mobile experience, new
features, bug fixes, performance improvements:

1. Go to github.com/ambhrni/shastra-adhyayana
2. Fork the repository
3. Create a branch for your change
4. Make your changes and test locally (see README for setup instructions)
5. Submit a pull request with a clear description of what you changed and why
6. The curator will review and merge if it meets the platform's standards

### Content contributions (texts, annotations, corrections)
If you want to contribute Sanskrit texts, commentary annotations,
nyāya concept definitions, or corrections to existing content:

1. Register at tattvasudha.org
2. Email tattvasudhaa@gmail.com with your background and what you
   wish to contribute
3. The curator will review your background and grant curator access
   if appropriate
4. All content contributions go through the curator approval workflow
   before appearing to learners

## Reporting errors in Sanskrit texts

Accuracy of the sacred texts is paramount. If you spot any error —
a Sanskrit spelling mistake, an incorrect word, a misattributed
commentary, or any inaccuracy — please report it using either method:

1. Inline flagging (fastest): On any study page, click the flag icon
   on the passage where you found the error. Describe the error briefly.
   This goes directly to the curator's review queue.

2. Email tattvasudhaa@gmail.com with:
   - The passage number and section name where the error appears
   - The incorrect text as it currently appears
   - What you believe the correct text should be
   - Your source or reasoning (reference to a printed edition,
     manuscript, or commentary is most helpful)

All reported errors are reviewed by the curator before any correction
is made.

## Standards for contribution

### Philosophical standards
- All content must be faithful to Mādhva Dvaita Vedānta siddhānta
- Commentary and annotation must follow the interpretations of
  Jayatīrtha, Rāghavendra Tīrtha, and the established paramparā
- No Advaita, Viśiṣṭādvaita, or other darśana interpretations
  should be presented as Madhva's tattvavAda positions
- When in doubt, consult the mūla text and its established commentaries

### Technical standards
- Follow the existing Next.js 14 / TypeScript / Tailwind CSS patterns
- All new database tables must follow the text-agnostic schema design
- Test on both desktop and mobile before submitting
- Sanskrit text must use proper Unicode Devanāgarī encoding

## What we are building together

Tattvasudhā aims to be the definitive platform for Madhva Dvaita
Vedānta śāstra study — rigorous, accessible, and growing. Currently
featuring vādāvalī by Jayatīrtha with commentaries of Rāghavendra
Tīrtha and Śrīnivāsa Tīrtha. Future texts planned: Nyāyasudhā,
Anuvyākhyāna, and more.

Every contribution — a bug fix, a corrected Sanskrit word, a new
nyāya definition — is itself a small jñānayajña.
